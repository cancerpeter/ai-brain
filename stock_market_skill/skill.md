# 股票行情分析 Skill

## Skill 目标

将自然语言股票行情请求路由到合适的插件能力，并统一返回结构化、可读的结果摘要。

## 支持意图

- 实时行情
  - A 股实时行情
  - 港股实时行情
- K 线
  - 日 K
  - 分时 / 分钟线
- 财经新闻
- 财经新闻汇总
- 榜单 / 市场信号
  - 涨跌幅榜
  - 净流入榜
  - 换手率榜
- 全市场条件筛选
- 长线候选筛选
- 中长线候选筛选
- 短线候选筛选
- 三市场长短线推荐工作流
- 汇率

## 路由规则

### 新闻

命中关键词：

- 新闻
- 快讯
- 财经消息
- 最新消息

路由到：

- `GetLiveNews`

### 新闻汇总

命中关键词：

- 新闻汇总
- 新闻整理
- 汇总新闻
- 各网站新闻

路由到：

- `GetDailyNewsDigest`

### 汇率

命中关键词：

- 汇率
- 人民币兑
- 美元兑
- 外汇

路由到：

- `GetExchangeRate`

### 日 K

命中关键词：

- 日K
- 日线
- K线

路由到：

- `GetKLineDayStockData`

### 分时 / 分钟线

命中关键词：

- 分时
- 分钟线
- 分钟K

路由到：

- `GetKLineTimeStockData`

### 港股实时行情

命中关键词或特征：

- 港股
- HK
- `.HK`
- 腾讯
- 小米
- 阿里港股

路由到：

- `GetHKStockData`

### 默认 A 股实时行情

未命中更明确规则时：

- 优先走 `GetSinaStockData`
- 后续可扩展失败回退 `GetQQStockData`

### 全市场筛选

命中关键词：

- 筛选
- 选股
- 条件股
- 符合条件

路由到：

- `ScreenStocks`

### 长线候选筛选

命中关键词：

- 长线
- 长期投资
- 中长线
- 价值投资

路由到：

- `ScreenLongTermStocks`

核心指标：

- 市值下限
- PE(TTM) 上限
- 适中换手率
- 主力净流入

### 中长线候选筛选

命中关键词：

- 中长线
- 中长期
- 稳健配置

路由到：

- `ScreenMidLongTermStocks`

核心指标：

- 新闻主题共振
- 估值不过热
- 主力净流入
- 中大市值
- 换手不过热

### 短线候选筛选

命中关键词：

- 短线
- 强势股
- 超短
- 短期交易

路由到：

- `ScreenShortTermStocks`

核心指标：

- 涨跌幅强度
- 高换手率
- 主力净流入
- 动量排序

### 三市场长短线推荐工作流

入口：

- `python3 -m stock_market_skill.report_workflow`

读取文档：

- `workflow_strategy.md`

工作方式：

- 先读取文档中的市场与策略配置
- 再按 A 股 / 港股 / 美股分别生成长线与短线推荐
- 输出推荐理由与入手时机

## 输出规范

所有结果统一采用三段式输出：

1. 一句核心结论
2. 关键字段
3. 简短补充说明

### 行情类最少字段

- 名称
- 代码
- 最新价
- 涨跌额
- 涨跌幅
- 更新时间

### K 线类最少字段

- 最近趋势摘要
- 最近若干条 OHLC 数据
- 数据不足时的优雅降级提示

### 新闻类最少字段

- 标题
- 内容摘要
- 时间
- 标签

### 新闻汇总类最少字段

- 标题
- 来源网站
- 时间
- 去重后的摘要列表

### 汇率类最少字段

- 币种
- 当前汇率
- 时间
- 简短解释

## 失败处理策略

1. 标的不明确时，返回候选提示，不瞎猜。
2. 参数缺失时，尽量补默认值。
3. 工具返回空数据时，给出清晰友好的说明。
4. 多工具都可能命中时，按规则优先级执行：
   - 新闻 / 汇率 / K 线 / 榜单
   - 港股实时行情
   - A 股实时行情默认策略
5. 保持路由结果结构化，方便上层 workflow / agent 继续处理。
